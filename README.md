## サービス概要
毎月の固定費（電気・ガス・水道などの公共料金、保険料、サブスクリプション、借金・ローンの返済など）を一元管理し、視覚的に可視化することで健全な資金管理をサポートするWebアプリケーションです。
支払額や支払日を一目で確認でき、貯金に対する引き落とし比率から使いすぎを警告することで、計画的な金銭管理を実現します。
「使ってはいけないお金」に手を付けることを防ぎ、一ヶ月の実質的な可処分所得を明確に把握できます。

## このサービスへの思い・作りたい理由
多くの人が「給料日前になぜかお金がない」という経験をしています。その原因の多くは、毎月確実に引き落とされる固定費を正確に把握できていないことにあります。

電気代、水道代、携帯料金、保険料、サブスクリプション、クレジットカードの引き落とし、ローン返済など、毎月自動的に引き落とされる支出は、一つひとつは大きくなくても積み重なると大きな金額になります。特に現代は、動画配信、音楽配信、クラウドストレージなどのサブスクリプションに加え、様々な保険やサービスが複雑に絡み合い、「気づいたら毎月数万円引き落とされていた」というケースが少なくありません。

このサービスは、自分が本当に「自由に使えるお金」がいくらなのかを明確にし、無意識の浪費を防ぐことで、経済的な不安を軽減し、より計画的な生活を送れるようサポートしたいという思いから生まれました。

## ユーザー層について
### ターゲットユーザー
1. **20代〜30代の社会人**
   - 複数の固定費支払いがあり、全体像を把握しきれていない層
   - クレジットカードを日常的に使用し、後払いの管理が曖昧になりがち
   - 奨学金返済やカーローンなど、複数の返済義務を抱えている可能性が高い

2. **家計管理を始めたい人**
   - 家計簿アプリは細かすぎて続かなかった経験がある
   - まずは「毎月確実に出ていくお金」だけでも管理したいと考えている
   - シンプルで継続しやすいツールを求めている

3. **経済的不安を感じている人**
   - 毎月の収支がマイナスになりがち
   - 貯金ができない原因を明確にしたい
   - 無駄な支出を削減したいが、何から手をつけていいかわからない

## サービスの利用イメージ
1. **登録・可視化**
   - ユーザーは毎月引き落とされる固定費（公共料金、保険、サブスク、ローンなど）を登録
   - ダッシュボードで支払日、支払額、残債（ローンの場合）を一覧表示
   - グラフやチャートで月間の支払い総額を視覚的に確認

2. **アラート・警告機能**
   - 収入（貯金額）に対する引き落とし比率を自動計算
   - 設定した閾値（デフォルトまたはカスタム）を超えた場合に警告を表示
   - 「今月使えるお金」を明確に提示

3. **行動変容**
   - 不要な固定費（サブスクなど）の発見・解約判断をサポート
   - クレジットカードの使いすぎを事前に防止
   - 返済計画の見直しや、支払いスケジュールの最適化

**得られる価値**
- 自分の財務状況を正確に把握できる
- 無意識の浪費を防ぎ、計画的な支出が可能になる
- 経済的不安が軽減され、精神的な余裕が生まれる

## サービスの差別化ポイント・推しポイント
### 既存サービスとの差別化
一般的な家計簿アプリ（MoneyForward、Zaim等）との違い:
- **特化型のシンプルさ**: 日々の支出記録ではなく、「毎月引き落とされる固定費」のみに特化
- **継続のハードルが低い**: 毎日入力する必要がなく、一度登録すれば自動で管理
- **心理的な警告機能**: 単なる記録ではなく、「使いすぎ」を積極的にアラート

### 推しポイント
1. **「使えるお金」の明確化**
   - 収入から固定費・返済を引いた「実質可処分所得」を即座に表示
   - 「今月あと○○円使える」が一目でわかる

2. **カスタマイズ可能な警告システム**
   - ユーザーごとに異なる価値観に対応（貯金重視派、ゆとり派など）
   - 自分で閾値を設定できる柔軟性

3. **固定費全体の可視化**
   - 公共料金、保険、サブスク、ローンなど、あらゆる固定費を一箇所で管理
   - 解約忘れ防止アラート（サブスクなど）
   - 使用頻度と費用の費用対効果分析（将来的な機能）

4. **視覚的なわかりやすさ**
   - 借金の減少推移グラフで返済のモチベーション維持
   - 固定費カテゴリ別の円グラフで無駄を発見しやすく
